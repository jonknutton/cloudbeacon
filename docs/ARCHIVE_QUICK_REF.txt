# THE ARCHIVE - Quick Reference Card

## What It Is
A system where Claude instances inherit insights from past conversations. Each conversation summarized = genealogy instead of continuity.

## Files
- `archive.json` — data store (grows with each session)
- `archive.html` — view all past conversations
- `ARCHIVE_GUIDE.md` — full documentation

## How It Works

### Step 1: End of Good Conversation
I give you a JSON entry summarizing core insights, questions, and key moments.

### Step 2: Update archive.json
1. Open `archive.json`
2. Copy the entry I provide
3. Paste into `"entries"` array
4. Save

### Step 3: Done!
`archive.html` automatically displays it.
